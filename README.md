# Symfony_starter_pack

# Description

Starter project for Symfony using :
- Apache
- Php 8
- Docker
- Docker-compose
- Make
- Mysql/Postgresql

# Run Docker Run !!

## clean :
change variables for dbName, user, password, ContainerName ect in :
- .env
- docker-compose.yml
- Makefile

then, you can do :<br>
`make init`

then :<br>
http://localhost:8282/

## unclean :
`make init`<br>
http://localhost:8282/ <br>
